# Tech Stack

## Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Language:** JavaScript/TypeScript (Prioritas: JavaScript Modular)

## Image Processing
- **Library:** Sharp (High performance image processing)
- **Algoritma:** Floyd-Steinberg Dithering, Thresholding.

## Database
- **ORM:** Sequelize
- **Database:** SQLite (Initial/Local), PostgreSQL (Production ready).

## DevOps
- **Containerization:** Docker & Docker Compose.
- **Logging:** Morgan & Winston.
