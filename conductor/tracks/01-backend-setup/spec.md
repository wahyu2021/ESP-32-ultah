# Specification: Initial Backend Setup

## Goal
Menyiapkan struktur folder proyek Node.js, inisialisasi Express, dan setup basic routing beserta keamanan API Key.

## Requirements
- Struktur folder modular (controllers, routes, middlewares, services).
- Implementasi middleware `X-API-KEY`.
- Inisialisasi `package.json` dengan dependency utama (express, dotenv, morgan).
