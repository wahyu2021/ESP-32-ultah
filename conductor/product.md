# Product Definition - IoT Smart Frame

## Visi
Membangun jembatan komunikasi (API Bridge) antara pengirim pesan (WA Bot) dan perangkat display E-Paper berbasis ESP32 untuk memberikan pengalaman interaktif yang romantis dan personal.

## Fitur Utama
1. **API Bridge:** Menerima teks dan gambar.
2. **BWR Image Processor:** Konversi gambar ke format E-Paper (Hitam, Putih, Merah).
3. **Queue System:** Manajemen antrean pesan agar perangkat selalu mendapatkan data terbaru.
4. **Telemetri:** Monitoring status baterai dan koneksi perangkat.
